# Sunbrella ☀️🌳

A tool for finding cooler, shaded walking routes.

## Inspiration

After moving to Atlanta and attending Emory, we quickly realized how intense the heat can be, especially during the spring and summer. This made us wonder if there was a way to prioritize shade instead of just distance when navigating. That idea led us to build Sunbrella, a tool that helps people find more comfortable walking paths by identifying routes with greater shade coverage.

## Features
- 🌳 Shade-optimized routes that prioritize cooler paths
- 📍 GPS navigation for real-time route guidance
- 🔊 Voice navigation for accessibility
- 📊 Shade scoring to compare how shaded different routes are

## Sunbrella was built using:
- HTML / CSS / JavaScript – frontend interface
- Google Maps API – route generation and mapping
- SunCalc – calculating sun position to estimate shade
- ElevenLabs – voice navigation
- GitHub + VS Code – development and collaboration

## Installation & Running Sunbrella

To run Sunbrella locally:
-    1.	Download or clone the repository
-    2.	If downloaded as a ZIP, unzip the folder
-    3.	Locate the file sunbrella.html
-    4.	Open sunbrella.html in your browser

The application will run directly in your browser.
